# Civil Engineering Web App

A Python-based web application for civil engineering calculations and design.

## Features
- Input-based calculation
- Live output updates
- Excel-integrated backend

## How to Run
1. Install Python  
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
