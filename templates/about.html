About This Project

This Singly Reinforced Beam Design Calculator is a web-based tool built with Flask, Python, and modern web technologies.

Technology Stack:

  • Backend: Flask (Python web framework)
  • Data Processing: Pandas for Excel file handling
  • Frontend: HTML5, CSS3 with Flexbox
  • Design: Responsive layout for all devices

Purpose:

To provide structural engineers with a quick and reliable tool for designing concrete beams with single reinforcement layers.
